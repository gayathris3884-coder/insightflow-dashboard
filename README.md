## 📌 About the Project

**InsightFlow** is an AI-powered analytics dashboard where users can upload any CSV dataset and instantly receive:

- 📊 Auto-generated **interactive charts** (trend analysis + category comparison)
- 🤖 **AI-generated insights** in plain English powered by Gemini API
- 📋 **Dataset preview** with outlier and missing value detection
- 📈 **KPI cards** showing total records, columns, missing values, and outliers

> Designed using **Google Stitch** (Google Labs AI UI tool) — a next-gen AI design tool that converts prompts into production-ready UI.

## 🧠 How It Works

1. User uploads a CSV file via the drag & drop modal
2. Backend analyzes the data — counts records, detects outliers, finds missing values
3. Charts auto-update with trend and category data
4. Gemini AI generates insights in natural language
5. User can export a full PDF report
